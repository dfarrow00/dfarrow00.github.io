Toon Tanks


Itch.io page

Toon Tanks is a short Action/Shooter game made using Unreal Engine 5. The aim of the game is to complete each level with as many points as possible. Gain points by destroying enemy tanks and turrets.

This project was made whilst following the GameDev.tv course "Unreal Engine 5 C++ Developer". Through this project, I gained a deeper understanding of Unreal Engine's, input handling systems, particle systems and the importance of child blueprint classes derived from C++ classes. Once the section was completed, I added some features of my own to further push my learning. These include multiple levels, additional enemy types using basic AI behavior, additional sound effects and music, a main menu system, scoring, and HUD widgets.
