Rooted Rebellion


GGJ Game Page

Rooted Rebllion is a 2D action boss battle style game that was developed in 48 hours for Global Game Jam 2023. The theme of the jam was "Roots". The game was developed as part of a team using Unity and Github branches to manage the collabortation process. My main programming responsibilities were the player movement, the axe combat system, and the main menu programming.
