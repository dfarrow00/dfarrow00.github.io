Rooted Rebellion


GGJ Game Page

Rooted Rebllion is a 2D action boss battle style game that was developed in 48 hours for Global Game Jam 2023. The theme of the jam was "Roots" and was developed using Unity. In my role as a programmer, I was responsible for implementing the player movement, axe combat system and the main menu system. I collaborated closely with my teammate, who created the player animations, to ensure a smooth and responsive player experience which resulted in the game recieving positive feedback.
