Dungeon Escape


Itch.io page

Dungeon Escape is a short and simple 3D puzzle game made using Unreal Engine 5. The aim of the game is to escape the dungeon by solving puzzles and passing obstacles.

This project was made whilst following the GameDev.tv course "Unreal Engine 5 C++ Developer". Working on this project allowed me to gain a strong understanding of Unreal Engine's core features and workflow. Once the section was completed, I added some features of my own to further push my learning. These include the green object outline to show when the object is grab-able, sound effects, moving pillars, cell door and key mechanics and the end game trigger and UI.
