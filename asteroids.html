Asteroids Remake | Java

This project was created using Java for the 'Game Programming' University module assignment. The aim of the assignment was to create a "Space" themed 2D video game where players take control of a space ship which they must use to overcome various kinds of enemies and obstacles. This project was my first dive into game programming without a game engine which taught me the basics of low level game development. Not only did I learn about game programming logic and mechanics, but I also gained experience in problem solving, vector math and time management. The game recieved a mark of 85/100.
